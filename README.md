# Amazon Clone
Build a simplified version of Amazon website from scratch using ReactJS.

## Features
- Full-Text Search
- Stripe Payment Platform
- Firebase Authentication
- Cloud Firestore (NoSQL Cloud Database)
- Responsiveness without CSS media query
